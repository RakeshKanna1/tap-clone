# SkillTap Academy - Premium Placement Accelerator Landing Page

A premium, modern, and responsive placement bootcamp accelerator landing page built as an educational tech portfolio frontend project. It is inspired by the visual design and interactive patterns of platforms like Scaler, Coursera, and Coding Ninjas.

---

## Project Overview

- **Project Name**: SkillTap Academy
- **Tagline**: Learn. Practice. Get Placed.
- **Concept**: High-end tech bootcamps with built-in interactive dashboard trackers, LeetCode-style problem solvers, ATS resume builders, AI-driven mock interviews, and automated hiring board panels.

---

## Key Features & Visual Upgrades

1. **Sticky Floating Glassmorphic Header**:
   - Modern floating bar aligned at the top of the viewport with a `backdrop-filter: blur(12px)` glass design.
   - Dynamic navbar scroll progress bar indicator indicating page depth.
   - Interactive Scrollspy highlighting sections as they cross the viewport.

2. **Admissions & Career Acceleration Dashboards**:
   - **Hero Dashboard**: Displays live placement progress trackers (e.g., Google interview status), coding streaks, mock scores, resume audits, and course syllabus progress.
   - **LeetCode Coding Section**: Features problem difficulty distribution stats charts (Easy, Medium, Hard), global rank trackers, and solved counters.
   - **ATS Resume Builder Preview**: Shows scoring metrics, suggestions checklists, missing skills audits, and keyword matching.
   - **Mock Interview AI metrics**: Displays Confidence, Speaking Speed (Optimal WPM checks), Eye Contact trackers, and overall Readiness percentage meters.
   - **Hiring Board Dashboard**: Details active recruitment drives, eligibility rules, and offer release notifications.

3. **Curriculum Course Deep Dives**:
   - Alternating Coursera-style showcase blocks highlighting backend Spring Boot code controllers and data prediction insights charts.
   - Enhanced dynamic course catalog grids displaying program levels, syllabus modules, capstone projects, and certified badges.

4. **Placement Timeline Roadmap**:
   - A step-by-step learning journey roadmap tracking students from Assessment, Learning, and Coding to Mock Interviews, Resume Reviews, and Placement Readiness.

5. **Mentors Board & Admission Validation**:
   - Mentor profiles featuring ex-FAANG engineers and design leads with "Book Session" buttons.
   - Admission form upgraded with college/university inputs, graduation year selectors, and cohort timing preferences with floating focus label transitions.

---

## Technical Architecture

- **Semantic HTML5 Markup**: Ensures search indexing and ARIA compliance.
- **Vanilla CSS3 Custom Variables**: Supports custom animations (e.g. scroll reveals, timeline underlines, and card hovers) and handles responsiveness down to 320px width.
- **ES6 JavaScript Modules**: Handles carousel transitions, interactive filters, counter animations, and form validation safely.
